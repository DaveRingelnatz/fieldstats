# fieldstats
NodeJS script for receiving important data of own node from carriotaField (IOTA load balancer)

INSTALLATION:

- login to your node
- create folder "fieldstats"
- enter commands:
  * cd fieldstats
  * npm install request
  * npm install parse-ini
- download file and save it as "fieldstats.js" on node
- enter command:
  * node fieldstats.js
- ENJOY
